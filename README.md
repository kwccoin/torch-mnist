# torch-mnist
MNIST classifier implemented in torch

## requirements
- torch
- nn
- optim
- xlua
- mnist

### for GPU
- cutorch
- cunn

### for GUI
- qttorch
- qtlua

## usage
- `th run.lua`
- if you want use GPU, `th run.lua -type cuda`
- for more detail, `th -h run.lua`

## GUI
- `qlua mnist_gui.lua`
